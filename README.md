# yuruly

ログイン不要で、サクッと日程調整できる現代的な日程調整サービスのMVPです。

## 特徴

- 🚀 **ログイン不要**: 面倒な会員登録なし、URLを共有するだけ
- ⏰ **柔軟な時間設定**: 終日・ざっくり（朝/昼/夕方/夜）・詳細時間から選択可能
- 📱 **モバイルファースト**: スマホで10秒以内に回答完了
- 🎨 **モダンなUI**: パステルカラーを基調とした親しみやすいデザイン
- ✨ **直感的な操作**: カレンダーから日付を選び、タップで◯△✕を切り替え

## 技術スタック

- **Frontend**: Next.js 14 (App Router), TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **Icons**: Lucide React
- **Date Handling**: date-fns

## セットアップ

### 1. 依存関係のインストール

```bash
npm install
```

### 2. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## 実装済みの機能

### ✅ イベント作成画面（幹事用）

- イベント名・説明の入力フォーム
- カレンダーUIで複数日の選択
- 各候補日に対する柔軟な時間設定
  - 終日（時間指定なし）
  - ざっくり枠（朝・昼・夕方・夜）
  - 詳細時間（開始時刻〜終了時刻）
- 選択中の候補日のリスト表示
- URL発行（回答用・管理用）
- Supabaseへのデータ保存

### ✅ 回答画面（参加者用）

- イベント情報の表示
- 候補日に対する◯△✕の回答
- タップで回答を切り替え
- 名前・コメントの入力
- 回答の送信

### ✅ 結果表示画面

- 回答者数の表示
- おすすめ日程（◯が多い順）
- 出欠表（マトリックス形式）
- 回答者一覧とコメント
- URLの共有機能

### ✅ 管理画面（幹事専用）

- イベント情報の確認
- 統計情報（回答者数、候補日数）
- 結果画面へのリンク
- イベントの削除機能
- 管理用トークンによる認証

## デザインコンセプト

- **カラーパレット**: パステル（紫・青・緑）を基調とした清潔感のあるデザイン
- **タイポグラフィ**: Nunitoフォントを使用した親しみやすい雰囲気
- **アニメーション**: 控えめなフェードイン・スケールアニメーションで心地よいUX
- **コンポーネント**: 角丸・シャドウを活かしたモダンなカードデザイン

## 使い方

### イベント作成の流れ

1. **イベント情報を入力**
   - イベント名を入力（必須）
   - 説明を入力（任意）

2. **候補日を選択**
   - カレンダーから複数の日付をクリック/タップ
   - 選択した日付に✅マークが表示される

3. **時間設定を調整**
   - 各候補日に対して時間設定を個別に選択
   - 「終日」「ざっくり」「詳細時間」から選択可能

4. **イベントを作成**
   - 「イベントを作成」ボタンをクリック
   - 回答用URLと管理用URLが発行される（予定）
